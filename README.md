# Cosmic Jumper - Платформер с 5 уровнями

## Описание игры
Cosmic Jumper - это космический платформер с красивой графикой и плавной физикой. Игрок должен собирать звезды и достигать порталов на каждом уровне.

## Особенности
- 5 уникальных уровней с разной сложностью
- Красивый космический дизайн
- Плавная физика движений и прыжков
- Система сбора звезд и подсчета очков
- Сохранение рекорда в localStorage
- Таймер игры
- Отзывчивый дизайн для всех устройств

## Управление
- **A / D** или **← / →** - движение влево/вправо
- **Space / W / ↑** - прыжок
- **P** - пауза/продолжить
- Кнопки управления в интерфейсе

## Уровни
1. **Стартовая зона** - обучение основам
2. **Прыжковые платформы** - цепочка платформ
3. **Мосты и пропасти** - требуется точность
4. **Вертикальный вызов** - вертикальный лабиринт
5. **Финальный уровень** - самое сложное испытание

## Как запустить
1. Скопируйте все файлы в одну папку
2. Откройте `index.html` в браузере через локальный сервер
   - Используйте Live Server в VS Code
   - Или `python -m http.server` в терминале
